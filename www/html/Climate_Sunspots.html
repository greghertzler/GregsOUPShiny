Sunspot Numbers

Description

World Data Center SILSO, Royal Observatory of Belgium

Format

csv file with 3316 rows and 2 columns

Details

  • DecimalYear: time variable in daily increments

  • Average: Simple average of the daily total sunspot number over all days of each calendar month.
