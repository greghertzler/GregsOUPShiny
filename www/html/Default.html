Default data

Description

Simulated data to estimate default parameters, rho=0.5, mu=15 and sigma=15, where rho is the rate of convergence, mu is the location and sigma is the scale.

Format

csv file with 6 rows and 2 columns

Details

  • tau: time variable

  • z: state variable
